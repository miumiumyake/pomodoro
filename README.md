# Pomodoro Robot 

Робот-помодоро на ESP8266 с анимированными глазами и сенсорным управлением.


## Управление

| Кнопка | Обычный режим | Work/Break режим |
|--------|---------------|------------------|
| **A** | Старт Pomodoro | Пауза / Продолжить |
| **B** | Смена эмоции | Выход (2 нажатия) |

## Схема подключения

| ESP8266 | TTP223 (A) | TTP223 (B) | OLED |
|---------|------------|------------|------|
| D5      | OUT        | -          | -    |
| D6      | -          | OUT        | -    |
| D1      | -          | -          | SCL  |
| D2      | -          | -          | SDA  |
| 3.3V    | VCC        | VCC        | VCC  |
| GND     | GND        | GND        | GND  |
